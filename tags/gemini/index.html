Gemini

My website and blog is available via Gemini

Gemini is a neat piece of technology. The Gemini protocol somewhat comparable to HTTP and the Gemtext file format comparable to HTML. But it’s simple. Like so simple that writing a client or server is actually doable. All popular web browsers (Chromium and its skins - and Firefox) have their roots in the 1990’s and there hasn’t been any new web browser becoming usable in many years. Not because people haven’t tried - I’m sure many people have - but because the standards around the modern web are so gigantic that it’s just not feasible to develop a web browser from scratch. ...
